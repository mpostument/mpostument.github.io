Sorted Squared Array

2022-02-14 :: Maksym Postument :: 2 min read (399 words)
#Algorithms  #Interview  #Codding Assignment 

Привіт!

В цьому завданні на вхід дається відсортований список, і його ціль полягає в тому щоб створити новий список який буде містити відсортовані числа у квадраті з першого списку. Найпростішим вирішення буде пройтись циклом по першому списку підняти всі числа у квадрат, додати в наш новий список і відсортувати. Але це рішення не буде оптимальним.

Давайте розглянемо оптимальніше рішення яке буде виконуватись лінійно.

Для прикладу візьмемо такий вхідний список - [-9, -1, 2, 3, 5, 6]

Для того, щоб вирішити це завдання за лінійний час, використаємо два вказівники які будуть вказувати на початок та кінець списку. Також нам потрібна змінна яка буде вказувати на яке місце в новому списку потрібно вставити число. Початково воно буде мати значення len(array) - 1 тому що ми будемо знаходити квадрат найбільшого числа і додавати його в кінець списку.

func SortedSquaredArray(array []int) []int {
	result := make([]int, len(array))
	leftIdx := 0
	rightIdx := len(array) - 1
	pos := len(array) - 1
}

Також як допоміжна функція нам потрібна функція обчислення абсолютного значення. В пакеті math така функція є. Але вона приймає тільки float64. Тому створимо свою

func Abs(x int) int {
	if x < 0 {
		return -x
	}
	return x
}

Тепер коли все готово потрібно створити цикл який буде працювати доки лівий вказівник не стане дорівнювати правому. В циклі будемо перевіряти чи елемент списку з вказівником rightIdx більший за leftIdx. Якщо умова виконується, то рахуємо квадрат числа під вказівником rightIdx і записуємо його в список result[pos] = array[rightIdx] * array[rightIdx], rightIdx потрібно зменшити нам один для того, щоб перейти до наступного елементу списку з правої сторони. Якщо ж умова не виконується, то це означає що число з списку під leftIdx є більшим, а отже записати в список потрібно його result[pos] = array[leftIdx] * array[leftIdx] і потрібно leftIdx збільшити на 1, для того, щоб перейти до наступного елементу списку з лівої сторони. І в кінці pos потрібно теж зменшити на один для того, щоб нове число записати в правильне місце списку.

func SortedSquaredArray(array []int) []int {
	result := make([]int, len(array))
	leftIdx := 0
	rightIdx := len(array) - 1
	pos := len(array) - 1
	
	for leftIdx <= rightIdx {
		if Abs(array[leftIdx]) < Abs(array[rightIdx]) {
			result[pos] = array[rightIdx] * array[rightIdx]
			rightIdx -= 1
		} else {
			result[pos] = array[leftIdx] * array[leftIdx]
			leftIdx += 1
		}
		pos -= 1
	}
 return result
}

Якщо викликати цей методі, то отримаємо результат [1, 4, 9, 25, 36, 81]
