Using Postgres With Golang

2022-02-20 :: Maksym Postument :: 3 min read (435 words)
#Go  #Postgres  #Beginner 

Привіт!

Сьогодні подивимось як за допомогою golang можна під’єднатися до postgres бази даних та виконувати запити. Для роботи з базою даних будемо використовувати github.com/jackc/pgx.

Розпочнемо з ініціалізації модуля та встановлення драйвера для бази.

go mod init
go get github.com/jackc/pgx/v4

Після цього pgx стане доступний для використання в коді.

Для тестової бази даних я використаю docker, але це може бути який тип інсталяції. Для цього потрібно створити файл docker-compose.yml з контентом:

version: '3'

services:
  postgres:
    image: postgres
    tty: true
    restart: always
    ports:
      - "5432:5432"
    environment:
      - POSTGRES_PASSWORD=postgres

І виконати команду docker-compose up -d. Docker-compose i docker мають бути встановленні. База даних готова для роботи, і тепер можна приступити до написання коду.

Розпочнемо з підключення до бази даних. Спочатку створюємо конфігурацію cfg в якій буде вказано адрес, порт, користувач, пароль та база даних до якої підключаємось. Оскільки в мене база даних працює локально то як хост я вказую localhost.Коли конфігурація готова викликаємо pgx.Connect(cfg) для створення підключення. Після закінчення роботи з базою підключення потрібно закрити тому додамо defer conn.Close() і перевіряємо помилки. Також щоб перевірити що база даних відповідає викличемо conn.Ping(context.Background())

package main

import (
	"context"
	"fmt"
	"log"
	"os"

	"github.com/jackc/pgx"
)

func main() {
	cfg := pgx.ConnConfig{
		Host:     "localhost",
		Port:     5432,
		User:     "postgres",
		Password: "postgres",
		Database: "postgres",
	}
	conn, err := pgx.Connect(cfg)
	if err != nil {
		fmt.Fprintf(os.Stderr, "Unable to connect to database: %v\n", err)
		log.Fatalf("Connect to database failed: %v\n", err)
	}
	defer conn.Close()

	err = conn.Ping(context.Background())
	if err != nil {
		log.Fatalf("Ping Failed: %v\n", err)
	}
}

Можна тепер почати робити запити. Для початку створимо нову таблицю і запишемо в неї дані. Для виклику запитів які нічого не повертають використовується метод Exec.

	_, err = conn.Exec(`
		CREATE TABLE IF NOT EXISTS users(
			id SERIAL PRIMARY KEY,
			name TEXT,
			email TEXT UNIQUE NOT NULL
		);
	`)
	if err != nil {
		log.Fatalln(err)
	}

	name := "TestUser1"
	email := "testuser@gmail.com"
	_, err = conn.Exec(`
		INSERT INTO users(name, email)
		VALUES ($1, $2);
	`, name, email)
	if err != nil {
		log.Fatalln(err)
	}

Дані створено, і можна спробувати їх прочитати. Якщо вертається з бази один запис, то можна використати метод QueryRow, якщо багато, то Query. Також для того, щоб прочитати дані потрібна структура, в нас це буде структура User яка буде мати такі самі параметри як і поля в таблиці.

type User struct {
	Id    int
	Name  string
	Email string
}

І сам запит щоб отримати дані

	user := User{}
	err = conn.QueryRow("SELECT * FROM users WHERE name=$1;", name).Scan(&user.Id, &user.Name, &user.Email)
	if err != nil {
		fmt.Println(err)
		return
	}
	fmt.Println(user)

Ось так за допомогою pgx легко можна читати та записувати дані. Pgx також може використовувати як драйвер до вбудованого sql пакета
