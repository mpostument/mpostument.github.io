Тепер архів потрібно розпакувати

sudo tar -C /usr/local -xzf go1.19.2.linux-amd64.tar.gz

Для того щоб команда go стала доступною її потрібно додати в PATH. Я використовую zsh тому шлях до go додам в файл ~/.zshrc

export PATH=$PATH:/usr/local/go/bin

Перезапустивши термінал команда go доступна

$ go version
go version go1.19.2 linux/amd64

Vscode install⌗

Для того щоб зручно було писати terraform код я використовую текстовий редактор Visual Studio Code. Після того як редактор встановлено для нього потрібно поставити Go додаток. Щоб встановити додаток потрібно натиснути Ctrl+Shift+X або натиснути на .
В пошуку ввести go, обрати Go та натиснути Install.

Для того щоб додаток go працював потрібно встановити додаткові go бібліотеки. Для цього потрібно натиснути Ctrl+Shift+P і написати go: install/update tools. Вибрати усі бібліотеки з списку та натиснути OK. В vscode буде написано коли всі пакети встановляться

Hello World⌗

Тепер створимо першу програму на go. Для початку створю директорію в які буде зберігатись код

mkdir -p ~/go/src/github.com/mpostument/hello_world

В теці hello_world спочатку потрібно ініціалізувати go модуль.

go mod init
go: creating new go.mod: module github.com/mpostument/hello_world

Як команда успішно закінчиться то з’явиться файл go.mod в якому буде міститься інформація про всі стороні бібліотеки які використовує модуль. Але зараз це файл містить тільки інформацію про версію go та назву модуля

module github.com/mpostument/hello_world

go 1.19

Тепер можна почати писати код. Створимо файл main.go і в нього додамо наступний код

package main

import "fmt"

func main() {
	fmt.Println("Hello World!")
}

Оскільки ми пишемо не бібліотеку а код який має викликатись тому package має бути main. Наступне імпорт бібліотек, fmt нам потрібна для того щоб вивести текст. Функція main викликається як ми запускаємо код, і в цій функції за допомогою методу Println з бібліотеки fmt виводимо текст в термінал.

Код можна запустити двома способами. Скомпілювавши бінарний файл або запустити за допомогою go run.

$ go run main.go
Hello World!

$ go build
$ ls
go.mod  hello_world  main.go
$ ./hello_world
Hello World!
