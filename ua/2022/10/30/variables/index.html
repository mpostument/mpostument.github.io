Привіт!

Поринемо у світ геймдеву з Game Maker 2. І сьогодні розпочнемо з параметрів в Game Maker. Створимо новий проект вибравши New Blank і назву його variables.

Object⌗

Параметри можна додавати до об’єктів. Об’єктом може бути практично будь чим починаю від героя і ворогів закінчуючи об’єктами які керують якоюсь логікою в грі. Для того щоб створити об’єкт потрібно натиснути правою кнопкою миші в Assets Browser на теці Objects і обрати Create -> Object.

Буде створено об’єкт з назвою Object1. Його потрібно одразу перейменувати. Рекомендується до назви додавати префікс, тому що якщо у нас буде об’єкт з назвою hero і ми захочемо додати спрайт який теж буде називатись hero отримаємо помилку що файл з такою назвою вже існує. Тому кожен об’єкт має мати префікс obj_. Таким чином об’єкт який я створю буде називатись obj_vars.

Після створення об’єкт одразу відкриється. Тут можна вказати спрайт для об’єкта, вказати чи він буде видимим та ще декілька опцій. Зараз нас цікавить вікно events. Саме тут ми будемо писати код який керуватиме нашим об’єктом.

Натиснемо кнопку Add Event та оберемо Create. Event типу Create викликається в момент створення об’єкту і тут зручно задавати якісь початкові характеристики. Натиснемо два рази лівою кнопкою миші щоб відкрити редактор коду.

Local Variables⌗

В редакторі створимо параметрів heroExp який буде відповідати за досвід нашого об’єкта. Цей параметри бути типу integer. Також додамо параметр heroName який буде типу string, і параметр heroAlive який буде типу bool

heroExp = 50
heroName = "Maksym"
heroAlive = true

Тепер додамо інший тип event який називається draw. Він відповідає за відображення об’єкту. Для цього знову натиснемо Add Event і в списку оберемо Draw -> Draw. Відкрию редактор для цього типу події і в коді напишу наступне

draw_text(20, 20, heroExp)
draw_text(20, 40, heroName)
draw_text(20, 60, heroAlive)

draw_text функція використовується для відображення тексту, перші два параметри це координати x так y, і третій параметр це текст який потрібно відобразити. Добавимо тепер наш об’єкт в кімнату і спробуємо і подивимось що буде.

Add object to room⌗

В Assets Browser потрібно відкрити теку Rooms і два рази натиснути мишкою на Room1. Відкриється вікно кімнати. В верху зліва має бути обрано Instances для того щоб можна було додавати об’єкти в кімнату. Тепер наш об’єкт можна просто перетягнути в кімнату. Також варто поміняти колір фону для того щоб було видно текст який відобразить нащ об’єкт. Для цього в опціях кімнати потрібно вибрати Background і вибрати колір (опція Colour). Я вибрав чорний колір, так як текст буде біло кольору.

Тепер коли все готово натиснемо F5 щоб запустити гру.

Dynamically change variables value⌗

Спробуємо тепер додати event який буде змінювати значення наших параметрів. Додамо event Key Press - Up який буде викликатись кожен раз як буде натиснута кнопка Up на клавіатурі і event Key Press - Space. Кнопка up буде додавати досвід нашому об’єкту а кнопка space буде вбивати наш об’єкт.

В код Key Press - Up додамо наступне

heroExp++

А в Key Press - Space

heroAlive = false

І запустимо гру. Якщо натиснути Up чи Space то параметри будуть змінені.

Global Vars⌗

Локальні параметри доступні тільки в межах одного об’єкту. Якщо потрібно додати параметр до якого буде мати доступ усі об’єкти то перед назвою параметра потрібно додати global.. Наприклад

global.heroAlive = true

і тепер цей параметр можна змінювати або використовувати в інших об’єктах
