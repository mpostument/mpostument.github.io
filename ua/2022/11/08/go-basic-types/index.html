Basic Types in Go

Привіт!

В цьому пості розглянемо базові типи даних в golang.

Bool⌗

Перший тип який розглянемо це bool. В цьому типу можливі лише два значення це true та false. Створити параметр типу bool можна в один за наступних способів:

	var isActive bool
	var isEnabled = false
	isLoaded := true

В другому і третьому варіанті ми одразу задаємо значення яке буде мати параметр. В першому ми нічого не вказуємо отже буде використовуватись нульове значення. Для типу bool це буде false. Виведу тип і значення кожного параметра

package main

import "fmt"

func main() {
	var isActive bool
	var isEnabled bool = false
	isLoaded := true

	fmt.Printf("%v: %T\n", isActive, isActive)
	fmt.Printf("%v: %T\n", isEnabled, isEnabled)
	fmt.Printf("%v: %T\n", isLoaded, isLoaded)
}

$ go run main.go
false: bool
false: bool
true: bool

Int⌗

Наступний це int. В golang є декілька типів int, також є unsigned int. Signed int може включати в себе числа з знаком мінус, в unsigned тільки плюсові числа. Відповідно всі типи int в golang uint8, uint16, uint32, uint64, int8, int16, int32 and int64.. Відрізняється вони тим який рендж цифр ми можемо використати. Наприклад для int8 цей рендж буде -128 to 127 а для uint це буде 0 to 255 і з кожним наступним типом цей рендж буде збільшуватись. Створюються параметри так само як і bool типу

	var a int64
	var b int
	var l uint = 20
	var c = 35
	d := 86

Тепер виведу кожне значення та тип

package main

import "fmt"

func main() {
	var a int64
	var b int
	var l uint = 20
	var c = 35
	d := 86

	fmt.Printf("%v: %T\n", a, a)
	fmt.Printf("%v: %T\n", b, b)
	fmt.Printf("%v: %T\n", l, l)
	fmt.Printf("%v: %T\n", c, c)
	fmt.Printf("%v: %T\n", d, d)
}

go run main.go
0: int64
0: int
20: uint
35: int
86: int

Для int нульовим значенням буде 0.

Над типом int можна проводити математичні операції. Але це має бути однаковий тип int. Якщо в нас є int64 та int32 ми не зможемо здійснити математичні операції. В цьому випадку спочатку потрібно буде привести їх до одного типу.

З математичних операцій доступно: додавання(+), віднімання(-), множення(*), ділення(/), залишок(%). Більшість операцій зрозумілі. Додаткової уваги потребує тільки ділення та залишок. Оскільки в int немає чисел після коми. То здійснивши ділення 10 на 3 ми отримаємо ціле число 3. і залишається залишок 1. Щоб отримати залишок якраз і використовується операція %

package main

import "fmt"

func main() {
	var a int = 10
	var b int = 3

	fmt.Println(a + b)
	fmt.Println(a - b)
	fmt.Println(a * b)
	fmt.Println(a / b)
	fmt.Println(a % b)
}

$ go run main.go
13
7
30
3
1

Float⌗

Числа з комою створюються схоже до int і мають такі ж самі математичні операції окрім обчислення залишку(%).

package main

import "fmt"

func main() {
	var a float32 = 10
	var b float32 = 3
	c := 10.4
	var d = 10.2
	fmt.Printf("%f: %T\n", a, a)
	fmt.Printf("%f: %T\n", b, b)
	fmt.Printf("%f: %T\n", c, c)
	fmt.Printf("%f: %T\n", d, d)
}

$ go run main.go
10.000000: float32
3.000000: float32
10.400000: float64
10.200000: float64

І математичні операції

package main

import "fmt"

func main() {
	var a float32 = 10.2
	var b float32 = 3.4

	fmt.Println(a + b)
	fmt.Println(a - b)
	fmt.Println(a * b)
	fmt.Println(a / b)
}

$ go run main.go
13.6
6.7999997
34.68
2.9999998

String⌗

Останній тип на сьогодні це string.

package main

import "fmt"

func main() {
	var a string = "Hello World"
	var c = "Hello"
	b := "World"

	fmt.Printf("%s: %T\n", a, a)
	fmt.Printf("%s: %T\n", b, b)
	fmt.Printf("%s: %T\n", c, c)
}

$ go run main.go
Hello World: string
World: string
Hello: string

Для string є одна математично операція це +. Вона дозволяє додавати два параметра типу string.

package main

import "fmt"

func main() {
	var c = "Hello "
	b := "World!"

	fmt.Printf(c + b)
}

$ go run main.go
Hello World!

Також до типу string можна звертатись за допомогою індексів. Можна представити що параметр типу string являє собою список, в golang нумерація в списку починається з нуля. Таким чином можна з string отримати тільки її частину. Наприклад

package main

import "fmt"

func main() {
	var c = "Hello "

	fmt.Printf("%v", string(c[2]))
}

$ go run main.go
l

Таким чином я отримую третій символ з параметра з назвою c. Але окрім цього ще потрібно конвертувати отримане значення в тип string. Інакше я б отримав число 108, число яке в UTF означає l.

На відміну від списків я не можу замінити якийсь символ в string. Тому що string являє собою незмінний тип даних. Але я може перетворити string в список байтів

import "fmt"

func main() {
	var c = "Hello "
	b := []byte(c)

	fmt.Println(b)
}

$ go run main.go
[72 101 108 108 111 32]

І тоді я зможу замінити будь яке значення в string

package main

import "fmt"

func main() {
	var c = "Hello "
	b := []byte(c)
	b[0] = byte(122)
	fmt.Println(string(b))
}

go run main.go
zello
