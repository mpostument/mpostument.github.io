Game Maker - Loops

2022-11-13 :: Maksym Postument :: 2 min read (401 words)
#GameMaker  #Beginner  #GameDev 

Table of Contents

  • Do until loop
  • Repeat

Привіт!

Тема сьогоднішнього постав це цикли. Розглянемо такі цикли як while, until та repeat. For цикл ми розглядали в темі про масиви, почитати про них можна тут.

While loop⌗

While цикл працює доки умова повертає true і має такий вигляд. Після ключового слова while в круглих дужках йде умова, і доки ця умова повертає true то цикл буде продовжуватись. Як тільки вона стане false цикл одразу закінчується

while (<expression>)
{
    <statement>;
    <statement>;
    ...
}

Приклад while в Game Maker. Спочатку створюю параметр i який має значення 0, і далі починаю цикл в якому вказую що він має працювати доки i є менше 5. В циклі за допомогою draw_text виводжу значення i на екран і збільшую i на одиницю. Якщо ж цього не зробити то i так і буде мати значення 0 а наш цикл ніколи не закінчиться і гра не запуститься. Цей код я добавив в Draw Event.

i = 0
while (i < 5){
	draw_text(10+70*i, 10, i)
	i++
}

Do until loop⌗

Наступний цикл це do until. На відміну від while він працює доки умова повертає false, як тільки вона стане true то цикл одразу закінчиться

do
{
    <statement>;
    <statement>;
    ...
}
until (<expression>);

І одразу розглянемо приклад, результат виконання цього циклу буде такий ж самий як і попередній з while. Але умова тут інша. В нас знову ж є i яке иає значення. Тоді ми починаємо цикл в якому відображаємо на екрані значення i і збільшуємо значення i, а далі в нас відбувається перевірка чи є i більша за 5. Якщо так то завершуємо цикл, а якщо ні то продовжуємо.

i = 0

do {
	draw_text(10+70*i, 10, i)
	i++
} until (i > 5)

Особливістю цього циклу є що навіть якщо i початково має значення більше за 5. Наприклад 6 то цикл відпрацює один раз, оскільки перевірка чи i є більшою за 5 відбувається в кінці циклу, і таким чином цикл встигає пройти одну ітерацію.

Repeat⌗

Останній вид циклу це repeat. На відміну від попередніх варіантів тут немає ніяких умов а ми просто вказуємо скільки раз пройтись по циклу

repeat (<expression>)
{
    <statement>;
    <statement>;
    ...
}

Спробуємо реалізувати теж саме що реалізувати за допомогою while та do циклів. Після ключового слова repeat в дужках вказуємо скільки раз має повторитись цикл а в самому тілі виконуємо такий самий код як і раніше, виводимо значення i і збільшуємо його на одиницю

i = 0

repeat(5) {
	draw_text(10+70*i, 10, i)
	i++
}
