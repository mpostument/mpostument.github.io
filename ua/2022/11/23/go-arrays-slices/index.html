Привіт!

В цьому пості розглянемо arrays та slices в golang.

Arrays⌗

Creation⌗

Масив в golang створюється наступним чином [NUMBER_OF_ELEMENTS_IN_ARRAY]TYPE{ELEMENTS}.

package main

import (
	"fmt"
)

func main() {
	students := [3]string{"John", "Lucy", "Alex"}
	fmt.Println(students)
}

$ go run main.go
[John Lucy Alex]

Щоб не вказувати число яке буде означати кількість елементів в масиві, можна вказати .... В цьому випадку go автоматично визначить скільки є елементів і створить масив такої довжини.

package main

import (
	"fmt"
)

func main() {
	students := [...]string{"John", "Lucy", "Alex"}
	fmt.Println(students)
}

Коли ми створюємо масив то не обов’язково при його створенні вказувати елементи які будуть в ньому. Можна створити пустий а додати елементи пізніше. Але в цьому випадку потрібно вказувати скільки елементів має бути. Тому що якщо ми вкажемо ... то створиться масив з довжиною 0 і додати більше ніяких елементів ми не зможемо.

package main

import (
	"fmt"
)

func main() {
	students := [...]string{}
	students[0] = "John"
	fmt.Println(students)
}

Якщо ми вкажемо кількість елементів але не вкажемо їх при створенні то будуть використані нульові значення даного типу. Наприклад наш масив буде типу int то він весь буде заповнений нулями.

package main

import (
	"fmt"
)

func main() {
	studentsGrades := [3]int{}
	fmt.Println(studentsGrades)
}

$ go run main.go
[0 0 0]

Change values⌗

Тепер коли в нас є пустий масив ми можемо додавати в нього значення. Робиться це наступним чином arrayName[INDEX] = VALUE. Індекси починаються з нуля, якщо нам потрібно додати перший елемент то індекс його буде рівний 0, індекс другого елемента буде рівний 1 і так далі

package main

import (
	"fmt"
)

func main() {
	studentsGrades := [3]int{}
	studentsGrades[0] = 99
	studentsGrades[2] = 22
	fmt.Println(studentsGrades)
}

$ go run main.go
[99 0 22]

В цьому прикладі ми вказали значення для першого елемента а третього, а другий елемент залишився без змін.

Подивитись довжину списку можна методом len()

package main

import (
	"fmt"
)

func main() {
	studentsGrades := [3]int{}
	fmt.Println(len(studentsGrades))
}

$ go run main.go
3

Copy array⌗

Масив можна скопіювати, для цього просто потрібно присвоїти його іншому параметру. Наприклад

package main

import (
	"fmt"
)

func main() {
	studentsGrades := [3]int{1, 2, 3}
	b := studentsGrades
	fmt.Printf("Students grades: %v\n", studentsGrades)
	fmt.Printf("Array b: %v\n", b)
}

Спробуємо тепер змінити значення в масиві b і вивести значення

package main

import (
	"fmt"
)

func main() {
	studentsGrades := [3]int{1, 2, 3}
	b := studentsGrades
	b[0] = 99
	fmt.Printf("Students grades: %v\n", studentsGrades)
	fmt.Printf("Array b: %v\n", b)
}

$ go run main.go
Students grades: [1 2 3]
Array b: [99 2 3]

Як бачимо змінився тільки другий масив, перший залишився без змін. А це тому що ми зробили копію масиву. Для того щоб передати в інший параметр той самий масив а не його копію потрібно використати &. В майбутніх постах ми ще детальніше розглянемо що це таке а зараз просто спробуємо його використати

package main

import (
	"fmt"
)

func main() {
	studentsGrades := [3]int{1, 2, 3}
	b := &studentsGrades
	b[0] = 99
	fmt.Printf("Students grades: %v\n", studentsGrades)
	fmt.Printf("Array b: %v\n", b)
}

go run main.go
Students grades: [99 2 3]
Array b: &[99 2 3]

Slices⌗

Creating⌗

Слайси є дуже схожими до масивів, вони навіть створюються дуже схоже. Єдине що в слайсів ми наперед не задаємо скільки елементів буде с слайсі.

package main

import (
	"fmt"
)

func main() {
	sliceOne := []int{1, 2, 3}
	fmt.Printf("Slice One: %v\n", sliceOne)
}

$ go run main.go
Slice One: [1 2 3]

Також можна подивитись довжину слайсу за допомогою метода len(). Але також до метода len в нас додається метод cap.

package main

import (
	"fmt"
)

func main() {
	sliceOne := []int{1, 2, 3}
	fmt.Printf("Len: %v, Capacity: %v\n", len(sliceOne), cap(sliceOne))
}

$ go run main.go
Len: 3, Capacity: 3

Append to slice⌗

Що ж це означає? Слайси використовуються масиви для зберігання даних. І capacity вказує розмір масиву який зараз використовує слайс.

Основною відмінністю є те що ми можемо розширювати слайси (чого ми не можемо зробити з масивами). Наприклад я створюю слайс з трьох елементів, в цьому випадку len i cap буде рівна трьом. Спробуємо додати четвертий елемент в слайс. Для цього потрібно використати метод append.

package main

import (
	"fmt"
)

func main() {
	sliceOne := []int{1, 2, 3}
	fmt.Printf("Len: %v, Capacity: %v\n", len(sliceOne), cap(sliceOne))
	sliceOne = append(sliceOne, 22)
	fmt.Printf("Len: %v, Capacity: %v\n", len(sliceOne), cap(sliceOne))
}

$ go run main.go
Len: 3, Capacity: 3
Len: 4, Capacity: 6

Спочатку в нас len та cap були рівні трьом. Після того як додали один елемент то len збільшився на 1 і став 4, а от cap збільшився в два рази і став 6. Тому що не вигідно go кожен раз створювати новий масив і копіювати туди дані, тому go сам визначає який масив створити щоб не займати багато пам’яті але і не створювати новий масив що разу як ми додаємо новий елемент в слайс

За допомогою append можна додавати багато елементів до слайса SLICE_NAME = append(SLICE_NAME, ELEM1, ELEM2, ELEM3). Також можна додати слайс до слайса. Але це не спрацює якщо ми в методі append вкажемо слайс замість елемента.

package main

import (
	"fmt"
)

func main() {
	sliceOne := []int{1, 2, 3, 5, 10}
	sliceTwo := []int{11, 233, 34}
	sliceOne = append(sliceOne, sliceTwo)
	fmt.Printf("Slice one: %v\n", sliceOne)
	fmt.Printf("Slice two: %v\n", sliceTwo)
}

$ go run main.go
# command-line-arguments
./main.go:10:30: cannot use sliceTwo (variable of type []int) as type int in argument to append

Для того щоб могти додати слайс потрібно розпакувати, це можна зробити за допомогою ...

package main

import (
	"fmt"
)

func main() {
	sliceOne := []int{1, 2, 3, 5, 10}
	sliceTwo := []int{11, 233, 34}
	sliceOne = append(sliceOne, sliceTwo...)
	fmt.Printf("Slice one: %v\n", sliceOne)
	fmt.Printf("Slice two: %v\n", sliceTwo)
}

$ go run main.go
Slice one: [1 2 3 5 10 11 233 34]
Slice two: [11 233 34]

Також на відміну від масивів коли ми присвоюємо його іншому параметрі в нас не створюється копія а передається сам слайс

package main

import (
	"fmt"
)

func main() {
	sliceOne := []int{1, 2, 3}
	b := sliceOne
	b[1] = 999
	fmt.Printf("Slice one: %v\n", sliceOne)
	fmt.Printf("Slice b: %v\n", b)
}

$ go run main.go
Slice one: [1 999 3]
Slice b: [1 999 3]

Slice from slice⌗

Ми можемо робити слайс з існуючого слайсу за допомогою індексів ARRAY_NAME[first_index:last_index]. Наприклад в нас є слайс з 5 елементів і нам потрібно створити слайс який буде мати тільки останні три елемента, то це можна зробити наступним чином

package main

import (
	"fmt"
)

func main() {
	sliceOne := []int{1, 2, 3, 5, 10}
	sliceTwo := sliceOne[2:]
	fmt.Printf("Slice one: %v\n", sliceOne)
	fmt.Printf("Slice two: %v\n", sliceTwo)
}

$ go run main.go
Slice one: [1 2 3 5 10]
Slice two: [3 5 10]

sliceTwo := sliceOne[2:] ми створюємо новий слайс який включає в себе всі елементи починаючи з другого індексу до останнього. Тому після : нічого вказувати більше не потрібно. Якщо потрібно то можна вказати останній елемент який буде включено.

Якщо ж ми спробуємо замінити елемент в другому слайсі то він і заміниться в першому, це тому що обидва слайси використовують один і той ж масив з даними

package main

import (
	"fmt"
)

func main() {
	sliceOne := []int{1, 2, 3, 5, 10}
	sliceTwo := sliceOne[1:3]
	sliceTwo[0] = 222
	fmt.Printf("Slice one: %v\n", sliceOne)
	fmt.Printf("Slice two: %v\n", sliceTwo)
}

$ go run main.go
Slice one: [1 222 3 5 10]
Slice two: [222 3]

Create slice with make⌗

Для створення слайсів може використовуватись методу make(TYPE, LEN, CAP). Capacity є не обов’язковим аргументом. Якщо ми використаємо make без capacity то це буде відповідати тому ж самому що ми створюємо пустий слайс []int{}.

package main

import (
	"fmt"
)

func main() {
	sliceOne := make([]int, 5)
	fmt.Printf("Slice one len %v, capacity %v\n", len(sliceOne), cap(sliceOne))
}

$ go run main.go
Slice one len 5, capacity 5

Якщо ми знаємо потрібний capacity то його можна одразу вказати, а тоді go не доведеться створювати нові масиви і переносити дані в них по мірі додавання об’єктів

package main

import (
	"fmt"
)

func main() {
	sliceOne := make([]int, 5, 100)
	fmt.Printf("Slice one len %v, capacity %v\n", len(sliceOne), cap(sliceOne))
}

$ go run main.go
Slice one len 5, capacity 100
