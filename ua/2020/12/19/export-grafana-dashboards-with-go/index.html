Export Grafana Dashboards With Go

2020-12-19 :: Maksym Postument
#Monitoring  #Go  #Devops 

Привіт!

Сьогодні подивимось як за допомогою go можна експортувати дашборди з Grafana. Для взаємодії з Grafana я використаю sdk. Sdk можна встановити за допомогою go get github.com/grafana-tools/sdk.

Почнемо писати код. На початку файлу буде package, імпорти та main функція

package main

import (
	"context"
	"encoding/json"
	"fmt"
	"log"
	"os"

	"github.com/grafana-tools/sdk"
)

func main() {

}
)

Додамо декілька параметрів:

	var (
		boardLinks []sdk.FoundBoard
		rawBoard   sdk.Board
		meta       sdk.BoardProperties
        err        error
        grafanaURL string
        apiKey     string
        directory  string
    )

В boardLinks будуть добавлятись ссилки на дашборди в Grafana. В rawBoard буде зберігатись дашборд. meta буде використовуватись для того щоб отримати ім’я дашборда і зберегти його з таким іменем в файл. directory директорія в яку будуть зберігатись дашборди. grafanaURL - Grafana url з портом в форматі http://127.0.0.1:3030 та apiKey для авторизації в Grafana. В grafanaUR, apiKey та directory потрібно внести своє значення.

Створюємо Grafana клієнт

	ctx := context.Background()

    c := sdk.NewClient(grafanaURL, apiKey, sdk.DefaultHTTPClient)

І тепер за допомогою клієнта можна робити запити до Grafana api. І перше що можна отримати це список ссилок на дашборди

	if boardLinks, err = c.Search(ctx); err != nil {
		log.Fatalln(err)
	}

Тепер як ссилки на дашборди отримано можна і отримати самі дашборди та записати їх в файл.

	for _, link := range boardLinks {
		if rawBoard, meta, err = c.GetDashboardByUID(ctx, link.UID); err != nil {
			log.Printf("%s for %s\n", err, link.URI)
			continue
		}
		rawBoard.ID = 0
		writeDashboardToFile(directory, rawBoard, meta.Slug)
	}

Циклом я проходжу по усіх ссилках на дашборди в slice boardLinks і на кожного з них викликаю метод GetDashboardByUID, якого в параметри передаю контекст та UID дашборда.

Також важливо щоб поле ID було відсутнє, інакше Grafana не дозволить зробити експорт дашбордів. Через то що в sdk.Board воно має тип uint та omitempty я можу дати йому значення 0 і це поле не буде добавлено.

ID              uint       `json:"id,omitempty"`

І в кінці викликається метод writeDashboardToFile який приймає три параметри, директорія де зберігати дашборд, сам дашборд, та ім’я дашборда яке можна отримати з параметра meta.

Що залишилось це створити метод writeDashboardToFile

func writeDashboardToFile(directory string, dashboard sdk.Board, name string, tag string) {
	var (
		err           error
		dashboardFile *os.File
		fileName      string
	)

	if _, err = os.Stat(directory); os.IsNotExist(err) {
		os.MkdirAll(directory, 0755)
	}

	fileName = fmt.Sprintf("%s/%s.json", directory, name)
	dashboardFile, err = os.Create(fileName)
	if err != nil {
		log.Printf("failed to create file for dashboard %s", fileName)
	}

	defer dashboardFile.Close()

	err = json.NewEncoder(dashboardFile).Encode(dashboard)
	if err != nil {
		log.Printf("failed to encode dashboard json to file %s", fileName)
	}
}

В цьому методі перевіряю чи існує директорія, якщо ж ні то створюю її. Далі в цій директорії я створюю файл з назвою дашборда і за допомогою json.NewEncoder записую json в файл.

Наступним кроком на основі даного коду можна створити cli наприклад за допомогою cobra чи іншої бібліотеки.
