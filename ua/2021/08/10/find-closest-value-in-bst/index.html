Find Closest Value in BST

2021-08-10 :: Maksym Postument :: 2 min read (350 words)
#Algorithms  #Interview  #Codding Assignment 

Привіт!

Сьогодні подивимось як знайти найближче значення в BST. На вхід дається бінарне дерево та число, і потрібно знайти найближче число до заданого. Наприклад маємо дерево:

    6
   / \
  4  10
 / \   \
2   5   23

Та число 13. Потрібно знайти число яке є найближчим до 13 в дереві. Для цього нам потрібно ще одне число (найближче значення), яке має велике значення, наприклад безкінечність.

Починаємо з кореня, в даному випадку він має значення 6. Від цілі (число 13) по модулі віднімаю 6 та отримую 7. Таку ж саму дію проводжу з теперішнім найближчим значення яке дорівнює безкінечності, безкінечність - 13 = безкінечність. Порівнюємо два числа 7 та безкінечність, 7 є меншим отже нове найближче число буде 6. Далі використаємо атрибути бінарного дерева. Оскільки число 13 яке ми шукаємо більше за корінь 6 можна повну відкинути ліву частину дерева. І залишиться тільки

    10
      \
       23

Та найближче число 6. Проводимо те ж саме з наступним елементом дерева яке дорівнює 10. По модулю віднімаю від 10 значення цілі. |10 - 13|= 3. Та порівнює це з теперішнім найближчим числом (13-6=7). 3 менше за 7. А отже нове найближче число 10.

Залишився останній елемент дерева. Знову від цілі віднімаю значення з дерева |23 - 10| = 13. І порівнюю з теперішнім найближчим числом (13-10=3). 13 є більше 3, тому найближчим числом залишається 10. Більше елементів в дереві немає а отже найближчим числом до 13 являється 10.

Приклад розв’язку на golang

package main

import (
	"fmt"
	"math"
)

type Tree struct {
	root *Node
}

type Node struct {
	key   int
	left  *Node
	right *Node
}

func main() {
	tree := Tree{
		&Node{
			key: 5,
			left: &Node{
				key: 4,
			},
			right: &Node{
				key: 10,
			},
		},
	}

	findClosestValue(tree, 3)
}
func findClosestValue(tree Tree, target int) {
	fmt.Println(findClosestValueHelper(tree.root, target, math.MaxInt32))
}

func findClosestValueHelper(node *Node, target int, closest int) int {
	if node == nil {
		return closest
	} else {
		if math.Abs(float64(target)-float64(closest)) > math.Abs(float64(target)-float64(node.key)) {
			closest = node.key
		}
		if target < node.key {
			return findClosestValueHelper(node.left, target, closest)
		} else if target > node.key {
			return findClosestValueHelper(node.right, target, closest)
		}
	}
	return closest
}
