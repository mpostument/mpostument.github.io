Validate Subsequence

2021-08-05 :: Maksym Postument :: 2 min read (243 words)
#Algorithms  #Interview  #Codding Assignment 

Привіт!

Першим вхідним параметром буде список з числами, другим буде також список з числами. Суть завдання полягає знайти чи є другий список підпослідовність першого.

Для цього нам буде потрібен поінтер який буде вказувати на елементи з другого списку. Почнемо рухатись по кожному елементу з першого списку і перевіряти чи дорівнює він елементу з другого списку на який вказує поінтер.

Для прикладу перший список 5, 3, 22, 25, 9, -5, 8, 7 і другий 3, 9, -5, 7. Поінтер вказує на 3

>3, 9, -5, 7

Починаємо перебирати числа з першого списку і порівнювати з поінтером. 3 не дорівнює 5 а отже переходимо до наступного елементу. 3 = 3 значить переносимо поінтер на один вправо

3, >9, -5, 7

Порівнюємо з наступним елементом з першого списку 22 != 9 25 != 9 9 = 9 знову співпадіння, і переносимо поінтер знову вправо

3, 9, >-5, 7

Далі перебираємо елементи першого списку -5 = -5 знову переносимо поінтер вправо і рухаємось по першому списку 8 != 7 7 = 7

Таким чином ми знайшли останній елемент з другого списку а отже він являється підпослідовність першого.

Приклад розв’язку на go

package main

import "fmt"

func main() {
	sequence := []int{5, 3, 22, 25, 9, -5, 8, 7}
	subsequence := []int{3, 9, -5, 7}
	fmt.Println(validateSubsequence(sequence, subsequence))
}

func validateSubsequence(sequence []int, subsequence []int) bool {
	subsequencePtr := 0

	for _, num := range sequence {
		if num == subsequence[subsequencePtr] {
			subsequencePtr++
		}
	}
	return subsequencePtr == len(subsequence)
}
