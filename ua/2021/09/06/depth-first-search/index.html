Depth First Search

2021-09-06 :: Maksym Postument :: 1 min read (178 words)
#Algorithms  #Interview  #Codding Assignment 

Привіт!

Сьогодні розглянемо Depth-first Search алгоритм. В нас є дерево і потрібно пройти по всіх його елементах з ліва на право. Візьмемо для прикладу дерево

         1 - 6 
           /    \
      2 - 4      10 - 6
         / \      \
    3 - 2  5 - 4 23 - 7

Отже в цьому дереві є 6 елементів. Починаємо з кореня і переходимо в ліву вітку до 4, в цієї вітки є два дочірні вітки, йдемо в ліву до 2, повертаємось назад до 4 і йдемо в праву до 5. Повертаємось до кореня і переходимо в праву частину до 10 і залишається останній елемент тож переходимо до нього. В результаті в нас виходить 6, 4, 2, 5, 10, 23.

Приклад алгоритму на golang

import "fmt"

type Node struct {
	Value    int
	Children []*Node
}

func main() {
	tree := Node{
		Value:    5,
		Children: []*Node{{Value: 3, Children: []*Node{{Value: 6}}}, {Value: 7, Children: []*Node{{Value: 8}}}},
	}
	fmt.Println(tree.DepthFirstSearch([]int{}))
}

func (n *Node) DepthFirstSearch(array []int) []int {
	array = append(array, n.Value)
	for _, child := range n.Children {
		array = child.DepthFirstSearch(array)
	}
	return array
}
