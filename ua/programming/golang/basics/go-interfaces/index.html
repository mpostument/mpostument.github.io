Interfaces in Go

2023-01-15Maksym Postument2 min read (316 words)
#Go  #Beginner 

Table of Contents

  • Video

Привіт!

Тема сьогоднішнього постав інтерфейси. Інтерфейси це тип який визначає набір методів.

type Printer interface {
	Print() string
}

Тип буде імпленетувати інтерфейс коли для нього буде створено метод Print. Як створювати методи для типу ми розглядали в пості про functions

type Book struct {
	Name   string
	Author string
}

type Money int

func (b Book) Print() string {
	return fmt.Sprintf("%s - %s", b.Author, b.Name)
}

func (m Money) Print() string {
	return fmt.Sprintf("You have %d dollars", m)
}

В цьому коді в нас є два типи Money i Book які імплементують інтерфейс Printer оскільки в них є метод Print.

Тепер як ми це все можемо використати? Ми можемо створити функцію яка як аргумент буде приймати тип Printer. А оскільки наші типи Money та Books імплементують Printer інтерфейс ми зможемо передати його як аргумент до цієї функції

func WriteLog(p Printer) {
	log.Println(p.Print())
}

І тепер в методі main це все викличемо

func main() {
	a := Book{
		Name:   "He who fight monsters",
		Author: "Shirtaloon",
	}
	m := Money(99999999999)
	WriteLog(a)
	WriteLog(m)
}

І як результат отримаємо наступне

$ go run main.go
2023/01/15 13:17:01 Shirtaloon - He who fight monsters
2023/01/15 13:17:01 You have 99999999999 dollars

Повний код буде мати наступний вигляд

package main

import (
	"fmt"
	"log"
)

type Printer interface {
	Print() string
}

type Book struct {
	Name   string
	Author string
}

type Money int

func (b Book) Print() string {
	return fmt.Sprintf("%s - %s", b.Author, b.Name)
}

func (m Money) Print() string {
	return fmt.Sprintf("You have %d dollars", m)
}

func main() {

	a := Book{
		Name:   "He who fight monsters",
		Author: "Shirtaloon",
	}
	m := Money(99999999999)
	WriteLog(a)
	WriteLog(m)
}

func WriteLog(p Printer) {
	log.Println(p.Print())
}

Інтерфейси можуть зменшити дублювання коду. Також вони допомагають у тестах, де ви можете замінити функцію, яка підключається до бази даних, спеціальними функціями лише для перевірки, яка не виконує реального підключення, але повертає результат так ніби підключення відбулось.
