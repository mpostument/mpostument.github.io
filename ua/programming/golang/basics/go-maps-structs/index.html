Привіт!

Тема сьогоднішнього поста це maps i structs. І розпочнемо ми з maps

Maps⌗

Create map⌗

Maps це структура даних в якій дані задаються парами, ключ та значення. В golang створюється наступним чином. map[KEY_TYPE]VALUE_TYPE.

import "fmt"

func main() {
	studentsGrades := map[string]int{"John": 50, "Alex": 70}
	fmt.Println(studentsGrades)
}

$ go run main.go
map[Alex:70 John:50]

Інший варіант створення map за допомогою слова make

import "fmt"

func main() {
	studentsGrades := make(map[string]int)
	fmt.Println(studentsGrades)
}

Add to map⌗

Додається новий елемент в map наступним чином MAP_NAME[KEY] = VALUE

package main

import "fmt"

func main() {
	studentsGrades := map[string]int{}
	studentsGrades["Susan"] = 99
	fmt.Println(studentsGrades)
}

$ go run main.go
map[Susan:99]

Якщо map створено за допомогою map, додати елементи в неї можна таким самим чином.

Delete from map⌗

Видаляється елемент за допомогою delete метода, delete(MAP_NAME, KEY)

package main

import "fmt"

func main() {
	studentsGrades := map[string]int{"John": 56, "Alex": 67}
	studentsGrades["Susan"] = 99
	fmt.Println(studentsGrades)
	delete(studentsGrades, "Alex")
	fmt.Println(studentsGrades)
}

$ go run main.go
map[Alex:67 John:56 Susan:99]
map[John:56 Susan:99]

Get element from map⌗

Отримати значення ключа можна таким чином MAP_NAME[KEY].

package main

import "fmt"

func main() {
	studentsGrades := map[string]int{"John": 56, "Alex": 67}
	fmt.Println(studentsGrades["John"])
}

$ go run main.go
56

Але якщо ми спробуємо вказати ключ який не існую, ми не отримаємо помилки, а отримаємо нульове значення

package main

import "fmt"

func main() {
	studentsGrades := map[string]int{"John": 56, "Alex": 67}
	fmt.Println(studentsGrades["Susan"])
}

$ go run main.go
0

Для того щоб перевірити чи існує ключ можна використати наступну конструкцію VARIABLE, ok := MAP_NAME[KEY]. Параметр ok буде мати значення true/false в залежності від того чи існує ключ.

package main

import "fmt"

func main() {
	studentsGrades := map[string]int{"John": 56, "Alex": 67}
	susanGrade, ok := studentsGrades["Susan"]
	fmt.Printf("Susan, %v %v\n", susanGrade, ok)

	johnGrade, ok := studentsGrades["John"]
	fmt.Printf("John, %v %v\n", johnGrade, ok)
}

$ go run main.go
Susan, 0 false
John, 56 true

Struct⌗

Struct це колекція полів які можуть бути різних типів.

Create struct⌗

Struct створюється таким чином type NAME struct {FIELDS}.

package main

import "fmt"

type Student struct {
	Name     string
	LastName string
	Grade    int
	Subjects []string
}

func main() {
	studentJohn := Student{
		Name:     "John",
		LastName: "Snow",
		Grade:    90,
		Subjects: []string{"Algebra", "Geometry "},
	}

	fmt.Println(studentJohn)
}

go run main.go
{John Snow 90 [Algebra Geometry ]}

Також можна окремо звернутись до конкретного поля struct STRUCT_NAME.FILED_NAME

fmt.Println(studentJohn.Name)

$ go run main.go
John

Embedding⌗

Go немає inheritance як в традиційних ООП мовах. Але go використовує composition. Розглянемо як це можна використати

package main

import "fmt"

type Animal struct {
	Name     string
}

type Bird struct {
	Animal
    Speed int
}

func main() {
	kiwi := Bird{}
	kiwi.Name = "Kiwi"
    kiwi.Speed = 30

	fmt.Println(kiwi.Name)
}

$ go run main.go
Kiwi

Але на відміну від inheritance тут Bird не є Animal, але Bird має Animal.

Якщо потрібно створити екземпляр структури як і попередньому прикладі, коли ініціалізуємо поля в момент створення то виглядати це буде так

package main

import "fmt"

type Animal struct {
	Name     string
}

type Bird struct {
	Animal
    Speed int
}

func main() {
	kiwi := Bird{
		Speed:    80,
		Animal: Animal{
			Name:     "Kiwi"
		}}

	fmt.Println(kiwi.Name)
}

Anonymous struct⌗

Можна створювати анонімні структури, без попереднього створення типу структури з описом полів

package main

import "fmt"

func main() {
	kiwi := struct {
		name  string
		speed int
	}{
		name:  "Kiwi",
		speed: 80,
	}
	fmt.Println(kiwi.name)
}

$ go run main.go
Kiwi

Такі структури зручно використовувати щоб організувати дані, які довго не живуть в програмі.

Copy struct⌗

Якщо взяти структуру і заасайнити її іншому параметру то буде створено копію структури

package main

import "fmt"

func main() {
	kiwi := struct {
		name  string
		speed int
	}{
		name:  "Kiwi",
		speed: 80,
	}
	b := kiwi
	b.name = "Emu"
	fmt.Println(kiwi)
	fmt.Println(b)
}

$ go run main.go
{Kiwi 80}
{Emu 80}

Для того щоб передати цю саму структуру потрібно додати символ &

package main

import "fmt"

func main() {
	kiwi := struct {
		name  string
		speed int
	}{
		name:  "Kiwi",
		speed: 80,
	}
	b := &kiwi
	b.name = "Emu"
	fmt.Println(kiwi)
	fmt.Println(b)
}

$ go run main.go
{Emu 80}
&{Emu 80}
