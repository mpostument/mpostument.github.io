Привіт!

Тема сьогоднішнього постав функції в golang.

Function⌗

У всіх попередніх постах ми неодноразово використовували функції. Наприклад fmt.Println() є функцією. Створимо найпростішу функцію. В golang завжди є функція main яка являється вхідна точкою в програму. І відповідно якщо ми хочемо викликати свою функції то додамо її в main функцію

package main

import (
	"fmt"
)

func main() {
	printText()
}

func printText() {
	fmt.Println("Hello")
}

Таким чином у нас є дві функції main та printText. Функція printText викликається в функції main. Та функція main яка викликається при старті програми. Зараз наша функція не приймає аргументів, спробуємо додати аргумент типу string в неї

package main

import (
	"fmt"
)

func main() {
	printText("Hello!")
}

func printText(msg string) {
	fmt.Println(msg)
}

Якщо функція приймає аргумент, його обов’язково потрібно вказати при виклику функції. Інакше буде помилка

$ go run main.go
# command-line-arguments
./main.go:8:2: not enough arguments in call to printText
        have ()
        want (string)

Можна передати більше одного аргументу, і якщо ці аргументи одного типу то тип можна вказати тільки один раз.

package main

import (
	"fmt"
)

func main() {
	printText("Message1", "Message2")
}

func printText(msg, second_message string) {
	fmt.Println(msg, second_message)
}

$ go run main.go
Message1 Message2

Якщо в нас є параметр в методі main і ми змінюємо його в іншій функції, то оригінальне значення не змінюється оскільки golang робить його копію.

package main

import (
	"fmt"
)

func main() {
	message := "Hello"
	printText(message)
	fmt.Println(message)
}

func printText(message string) {
	message = "Good Bye"
	fmt.Println(message)
}

$ go run main.go
Good Bye
Hello

Але якщо нам треба змінити оригінальне значення то можна передати pointer.

package main

import (
	"fmt"
)

func main() {
	message := "Hello"
	printText(&message)
	fmt.Println(message)
}

func printText(message *string) {
	*message = "Bye!"
	fmt.Println(*message)
}

$ go run main.go
Bye!
Bye!

Якщо ж потрібно передати багато параметрів одного типу можна використати ... синтаксис.

package main

import (
	"fmt"
)

func main() {
	printText("Hello", "Bye")
}

func printText(messages ...string) {
	for _, msg := range messages {
		fmt.Println(msg)
	}
}

$ go run main.go
Hello
Bye

В цьому випадку всі параметри які були передані в функцію будуть у вигляді слайсу.

Return value⌗

В усіх попередніх випадках ми виводили значення прямо в функції. Якщо ж нам потрібно передати значення в іншу функцію то можна використати ключове слово return. І в самій функції потрібно вказати який тип буде повернуто.

package main

import "fmt"

func main() {
	result := printText(6, 3)
	fmt.Println(result)
}

func printText(a, b int) int {
	return a / b
}

$ go run main.go
2

Також якщо потрібно повертати можна pointer.

package main

import (
	"fmt"
)

func main() {
	r := returnPointer()
	fmt.Println(*r)
}

func returnPointer() *int {
	result := 4 / 2
	return &result
}

$ go run main.go
2

Return multiple values⌗

З функції можна і вертати декілька значень.

package main

import (
	"fmt"
)

func main() {
	first, second := returnMultiple()
	fmt.Println(first, second)
}

func returnMultiple() (int, int) {
	result := 4 / 2
	secondResult := 6 / 2
	return result, secondResult
}

$ go run main.go
2 3

Return error⌗

Часто з функції окрім значення також вертають помилку. І після того в функції яка робить виклик перевіряється, чи була помилка.

package main

import (
	"fmt"
	"log"
)

func main() {
	r, err := returnMultiple(1, 0)
	if err != nil {
		log.Fatalln(err)
	}
	fmt.Println(r)
}

func returnMultiple(a, b int) (int, error) {
	if b == 0 {
		return 0, fmt.Errorf("Cannot divide be zero")
	}
	result := a / b
	return result, nil
}

Anonymous functions⌗

Анонімна функція це функція в якої немає іменні. Створюється вона так само як і звичайна функція тільки єдине що без іменні.

package main

import (
	"fmt"
)

func main() {
	func() {
		fmt.Println("Hello")
	}()
}

$ go run main.go
Hello

Для того щоб функція викликалась потрібно в кінці додати ще пару (). Інакше отримаємо помилку

$ go run main.go
# command-line-arguments
./main.go:8:2: func() {…} (value of type func()) is not used

Якщо ж функцію не потрібно одразу викликати, її можна призначити якомусь параметру і викликати коли потрібно

package main

import (
	"fmt"
)

func main() {
	a := func() {
		fmt.Println("Hello")
	}
	a()
}

$ go run main.go
Hello

Methods⌗

Метод це теж функція, але яка належить якомусь типу. Наприклад в нас є структура чи свій власний тип і для них ми можемо створити методи. В цьому прикладі у нас є структура Hello з двома полями. Далі ми створюємо функцію але додатково перед назвою функції вказуємо до якого типу водна відноситься, в нашому випадку це структура з назвою Hello і за допомогою літери h ми можемо звертатись до її полів. Після того як в методі main ми створили екземпляр ції структури ми можемо викликати наш метод за допомогою VARIABLE.METHOD_NAME()

package main

import (
	"fmt"
)

type Hello struct {
	Name    string
	Message string
}

func (h Hello) hello() {
	fmt.Println(h.Message, h.Name)
}
func main() {
	a := Hello{
		Name:    "Maksym",
		Message: "Hello",
	}
	a.hello()
}

$ go run main.go
Hello Maksym
