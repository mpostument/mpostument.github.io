Go Constants

Привіт!

Розглянемо що таке константи, як їх потрібно називати в golang, також подивимось що таке iota.

Naming⌗

Якщо ви знайомі з константами в інших мовах програмування то ви знаєте що їх назви зазвичай включають тільки великі літери, наприклад PORT_NUMBER. В golang так ми не можемо зробити оскільки зробивши першу літеру великою ми одразу зробимо константу доступною для інших пакетів. Тому в golang називаємо константи так само як і параметри portNumber або PortNumber якщо потрібно щоб константа було доступна в інших пакетах.

package main

import "fmt"

func main() {
	const portNumber int = 80
	fmt.Printf("%v: %T", portNumber, portNumber)
}

$ go run main.go
80: int

Так саме як і з параметрами тип можна не вказувати

const portNumber = 80

Особливістю константи є те що вона має мати значення на момент компіляції, наприклад якщо в константі я спробую підняти константу в квадрат я отримаю помилку, оскільки це обчислюється в момент запуску програми, а не на етапі компіляції

package main

import (
	"fmt"
	"math"
)

func main() {
	const portNumber float64 = math.Pow(2.0, 2.0)
	fmt.Println(portNumber)
}

$ go run main.go
# command-line-arguments
./main.go:9:29: math.Pow(2.0, 2.0) (value of type float64) is not constant

Shadowing⌗

Так само як і параметрами, shadowing також працює і з константами.

package main

import (
	"fmt"
)

const portNumber int32 = 80

func main() {
	const portNumber int = 8080
	fmt.Printf("%v: %T\n", portNumber, portNumber)
	printConstant()
}

func printConstant() {
	fmt.Printf("%v: %T\n", portNumber, portNumber)
}

$ go run main.go
8080: int
80: int32

Math Operations⌗

Так само і працюють математичні операції

package main

import "fmt"

func main() {
	const a int = 8080
	const b int = 10

	fmt.Printf("%v: %T\n", a+b, a+b)
	fmt.Printf("%v: %T\n", a-b, a-b)
	fmt.Printf("%v: %T\n", a*b, a*b)
	fmt.Printf("%v: %T\n", a/b, a/b)
}

$ go run main.go
8090: int
8070: int
80800: int
808: int

Якщо ж спробуємо додати константи різних типів то отримаємо помилку

package main

import "fmt"

func main() {
	const a int32 = 8080
	const b int = 10

	fmt.Printf("%v: %T\n", a+b, a+b)
}

$ go run main.go
# command-line-arguments
./main.go:9:25: invalid operation: a + b (mismatched types int32 and int)

Але тут є одна особливість, якщо в одній з констант ми не вкажемо тип а просто значення 8080, то компілятор автоматично визначить його як int. Тоді створимо іншу константу типу int16 і спробуємо їх додати, то отримаємо успішне виконання. Це тому що компілятор в момент компіляції заміняє константу її значенням у всіх місцях де вона використовується.

package main

import "fmt"

func main() {
	const a = 8080
	const b int16 = 10

	fmt.Printf("%v: %T\n", a, a)
	fmt.Printf("%v: %T\n", b, b)
	fmt.Printf("%v: %T\n", a+b, a+b)
}

Для компілятора цей код буде виглядати як

package main

import "fmt"

func main() {
	const a = 8080
	const b int16 = 10

	fmt.Printf("%v: %T\n", 8080, 8080)
	fmt.Printf("%v: %T\n", 10, 10)
	fmt.Printf("%v: %T\n", 8080+10, 8080+10)
}

І як результат ми отримаємо int16

$ go run main.go
8080: int
10: int16
8090: int16

Iota⌗

При створенні констант можна вказувати спеціальне слово iota

package main

import "fmt"

const (
	a = iota
)

func main() {
	fmt.Printf("%v: %T\n", a, a)
}

$ go run main.go
0: int

iota дозволяє створювати пронумеровані константи.

package main

import "fmt"

const (
	a = iota
	b = iota
	c = iota
)

func main() {
	fmt.Printf("%v: %T\n", a, a)
	fmt.Printf("%v: %T\n", b, b)
	fmt.Printf("%v: %T\n", c, c)
}

$ go run main.go
0: int
1: int
2: int

Слово iota для кожної константи не є обов’язковим, якщо ці константи в одному блоці

package main

import "fmt"

const (
	a = iota
	b
	c
)

func main() {
	fmt.Printf("%v: %T\n", a, a)
	fmt.Printf("%v: %T\n", b, b)
	fmt.Printf("%v: %T\n", c, c)
}

$ go run main.go
0: int
1: int
2: int

Часом потрібно пропустити значення в константі, найчастіше нульове якщо ми потім будемо це порівнювати, це можна зробити таким чином

package main

import "fmt"

const (
	_ = iota
	a
	b
	c
)

func main() {
	fmt.Printf("%v: %T\n", a, a)
	fmt.Printf("%v: %T\n", b, b)
	fmt.Printf("%v: %T\n", c, c)
}

$ go run main.go
1: int
2: int
3: int

Також iota може починатись не з нуля якщо до неї додати число

package main

import "fmt"

const (
	a = iota + 5
	b
	c
)

func main() {
	fmt.Printf("%v: %T\n", a, a)
	fmt.Printf("%v: %T\n", b, b)
	fmt.Printf("%v: %T\n", c, c)
}

$ go run main.go
5: int
6: int
7: int
