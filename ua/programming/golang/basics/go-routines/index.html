Привіт!

Тема сьогоднішнього поста goroutines. Ми розглянемо що це таке та як їх використовувати.

Goroutine⌗

Goroutine дозволяє запускати код одночасно. Наприклад ми можемо декілька функцій викликати в один той самий момент не очікуючи одна на одну. Для того щоб створити goroutine перед викликом функції додати ключове слово go.

package main

import "fmt"

func main() {
	go hello()
}

func hello() {
	fmt.Println("Hello!")
}

Є інший варіант, за допомогою WaitGroup.

WaitGroup⌗

WaitGroup дозволяє зачекати на виконання goroutine. Для цього створюємо параметр wg який має значення sync.WaitGroup{}. І надалі в коді перед викликом goroutine ми викликаємо wg.Add(1) щоб додати одну goroutine в список очікування. Після виклику goroutine потрібно додати wg.Wait() для того щоб зачекати на виконання. А в самій goroutine потрібно додати wg.Done() щоб забрати попередньо додану goroutine з списку очікування. І тепер коли я викличу код то вже отримаю результат.

package main

import (
	"fmt"
	"sync"
)

var wg = sync.WaitGroup{}

func main() {
	wg.Add(1)
	go hello()
	wg.Wait()
}

func hello() {
	fmt.Println("Hello!")
	wg.Done()
}

Mutex⌗

Спробуємо запустити дві goroutine які будуть одночасно змінювати/читати з одного й того ж параметра.
В цьому прикладі goroutine викликаються в циклі. Одна з функцій збільшує значення параметра counter на 1, а інша виводить значення.

package main

import (
	"fmt"
	"sync"
)

var wg = sync.WaitGroup{}
var counter = 0

func main() {
	for i := 0; i < 10; i++ {
		wg.Add(2)
		go readCounter()
		go increment()
	}
	wg.Wait()

}

func increment() {
	counter++
	wg.Done()
}

func readCounter() {
	fmt.Println(counter)
	wg.Done()
}

Як це можна вирішити? За допомогою Mutex. Ми можемо дозволити доступ до нашого об’єкту тільки для однієї goroutine. В прикладі використаємо RWMutex який дозволяє читати дані будь якій кількості goroutine а записувати тільки одній. Перед goroutine яка читає дані додаємо RLock а перед тою що записує додаємо Lock. В самій функції після того як відбувся запис/читання знімаємо lock. І тепер якщо запустити код то отримаємо очікуваний результат.

package main

import (
	"fmt"
	"sync"
)

var wg = sync.WaitGroup{}
var mutex = sync.RWMutex{}

var counter = 0

func main() {
	for i := 0; i < 10; i++ {
		wg.Add(2)
		mutex.RLock()
		go readCounter()
		mutex.Lock()
		go increment()
	}
	wg.Wait()

}

func increment() {
	counter++
	mutex.Unlock()
	wg.Done()
}

func readCounter() {
	fmt.Println(counter)
	mutex.RUnlock()
	wg.Done()
}
