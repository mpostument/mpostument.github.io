If and Switch in golang

Привіт!

Тема сьогоднішнього поста це If та Switch в go.

If⌗

Блок if виконується в тому випадку ящо умова повертає true.

package main

import "fmt"

func main() {
	if true {
		fmt.Println("Conditions is true")
	}

	if false {
		fmt.Println("Conditions is false")
	}
}

$ go run main.go
Conditions is true

В цьому випадку було виведено тільки Conditions is true тому що умова в другому if блоці була false. Розглянемо приклад більш наближений до реального життя.

package main

import "fmt"

func main() {
	firstNumber := 10
	secondNumber := 20

	if firstNumber > secondNumber {
		fmt.Println("First number is bigger")
	}

	if firstNumber < secondNumber {
		fmt.Println("First number is bigger")
	}

	if firstNumber == secondNumber {
		fmt.Println("First number is bigger")
	}
}

$ go run main.go
First number is bigger

Тут знову ж було виведено тільки перший рядок. Тому що усі інші повернули false. Так як перше число є більшим за друге. Якщо змінити числа то буде виведено інший результат.

Окрім вище вказаних < менше, > більше та == рівне я ще наступні оператори != не рівне, >= більше рівне та <= менше рівне.

Також в одному if блоці може бути декілька умов які можуть об’єднані && - і або || - чи. && буде рівним true тільки якщо всі умови повернуть true. При || повернеться true якщо одна з умов поверне true.

package main

import "fmt"

func main() {
	firstNumber := 10
	secondNumber := 20

	if firstNumber >= 20 && secondNumber >= 20 {
		fmt.Println("Both number higher then 20")
	}

	if firstNumber >= 20 || secondNumber >= 20 {
		fmt.Println("One or both number higher then 20")
	}

	if firstNumber != secondNumber {
		fmt.Println("Numbers are not equal")
	}
}

$ go run main.go
One or both number higher then 20
Numbers are not equal

В цьому прикладі виводиться друге повідомлення оскільки одне з чисел більше за 20. Також виведена останнє повідомлення тому що числа не є рівними. Для того щоб не всі умови перевірялись а наприклад зупинялись після першої яка повернула true можна застосувати if/else. Перепишемо той самий код з додаванням else.

package main

import "fmt"

func main() {
	firstNumber := 10
	secondNumber := 20

	if firstNumber >= 20 && secondNumber >= 20 {
		fmt.Println("Both number higher then 20")
	} else if firstNumber >= 20 || secondNumber >= 20 {
		fmt.Println("One or both number higher then 20")
	} else if firstNumber != secondNumber {
		fmt.Println("Numbers are not equal")
	} else {
		fmt.Println("Final case")
	}
}

$ go run main.go
One or both number higher then 20

В цьому випадку коли коли перша умова повернула true, наступні вже не перевіряються, тому бачимо тільки один вивід.

Switch⌗

Switch схожий на if умову але використовується тоді коли в нас є багато схожих перевірок.

package main

import "fmt"

func main() {
	number := 30

	switch number {
	case 10:
		fmt.Println("Number is 10")
	case 20:
		fmt.Println("Number is 20")
	case 30:
		fmt.Println("Number is 30")
	default:
		fmt.Println("Default cast")
	}
}

Декілька кейсів можна об’єднати в один

package main

import "fmt"

func main() {
	number := 30

	switch number {
	case 10, 20, 30:
		fmt.Println("Number is 10, 20 or 30")
	case 40, 50, 60:
		fmt.Println("Number is 40, 50 or 60")
	default:
		fmt.Println("Default cast")
	}
}

$ go run main.go
Number is 10, 20 or 30

Так само як і в if можна використовувати оператори порівняння. Виглядає switch так само, тільки біля нього не вказується назва параметру який ми перевіряємо.

package main

import "fmt"

func main() {
	number := 30

	switch {
	case number <= 30:
		fmt.Println("Number is less then 30")
	case number <= 60:
		fmt.Println("Number is less then 60")
	default:
		fmt.Println("Default cast")
	}
}

$ go run main.go
Number is less then 30

В цьому випадку не викликається наступний кейс де number <= 60 хоча він теж повертає true, щоб він викликався потрібно додати ключове слово, але тоді наступний кейс виконається не залежно яка в ньому умова

package main

import "fmt"

func main() {
	number := 30

	switch {
	case number <= 30:
		fmt.Println("Number is less then 30")
		fallthrough
	case number <= 10:
		fmt.Println("Number is less then 10")
	default:
		fmt.Println("Default cast")
	}
}

$ go run main.go
Number is less then 30
Number is less then 10

Незважаючи на то що number <= 10 вернуло false, Number is less then 10 всеодно було виведено.

Ще один варіант як використовується case це є перевірка типу.

package main

import "fmt"

func main() {
	var n any = 1

	switch n.(type) {
	case int:
		fmt.Println("N is int")
	case string:
		fmt.Println("N is string")
	default:
		fmt.Println("Undefined type")
	}
}

$ go run main.go
N is int

Якщо замінити 1 на “1” то буде виведено N is string.
