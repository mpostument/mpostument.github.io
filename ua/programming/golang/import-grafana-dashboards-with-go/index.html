Import Grafana Dashboards With Go

2020-12-23Maksym Postument2 min read (296 words)
#Monitoring  #Go  #Devops 

Привіт!

В попередньому пості я показував як можна експортувати дашборди з Grafana. Тепер подивимось як їх можна імпортувати в Grafana.

Як завжди на початку будуть імпорти та main функція

package grafana

import (
	"context"
	"encoding/json"
	"fmt"
	"io/ioutil"
	"log"
	"os"
	"strings"

	"github.com/grafana-tools/sdk"
)

func main() {

}

Додамо декілька параметрів:

	var (
		filesInDir []os.FileInfo
		rawBoard   []byte
        err        error
        grafanaURL string
        apiKey     string
        directory  string
    )

В filesInDir будуть зберігатись усі прочитані дашборди з файлової системи. А в rawBoard буде міститись сам контент файлів. directory директорія в яку будуть зберігатись дашборди. grafanaURL - Grafana url з портом в форматі http://127.0.0.1:3030 та apiKey для авторизації в Grafana. В grafanaUR, apiKey та directory потрібно внести своє значення.

Створюю Grafana клієнт за допомогою якого буду робити запити до Grafana Api

	ctx := context.Background()

    c := sdk.NewClient(grafanaURL, apiKey, sdk.DefaultHTTPClient)

Залишилось прочитати файли і можна їх експортувати.

	filesInDir, err = ioutil.ReadDir(directory)
	if err != nil {
		log.Fatal(err)
    }

Тепер циклом йду по усіх файлах. Але в директорії яку я читаю можуть бути іншій файли окрім .json тому буду перевіряти чи назва файлу закінчується на .json. Якщо файл все таки json записую його контент в rawBoard. І перетворюю цей файл в структуру sdk.Board. Також створюю об’єкт типу sdk.SetDashboardParams в якому вказую в яку папку імпортувати дашборд та чи перезаписати якщо такий дашборд вже існує. Коли всі параметри готові викликаю метод SetDashboard який і здійснить експорт.

	for _, file := range filesInDir {
		if strings.HasSuffix(file.Name(), ".json") {
			if rawBoard, err = ioutil.ReadFile(fmt.Sprintf("%s/%s", directory, file.Name())); err != nil {
				log.Println(err)
				continue
			}
			var board sdk.Board
			if err = json.Unmarshal(rawBoard, &board); err != nil {
				log.Println(err)
				continue
			}
			params := sdk.SetDashboardParams{
				FolderID:  sdk.DefaultFolderId,
				Overwrite: true,
			}
			_, err := c.SetDashboard(ctx, board, params)
			if err != nil {
				log.Printf("error on importing dashboard %s", board.Title)
				continue
			}
		}
    }
